Welcome to Liquid Carrot

Liquid Carrot is an open source AI framework library for Java Script developers

We specialize in creating neural network systems and deep learning models for structured (or unstructured) data sets

In short, we are aiming to make artifical intelligence more accessible to the world



Why are we using JavaScript?

JavaScript is a particularly great language for AI despite the naysayers
  1. FAQ with lot's of contributors and a large community
  2. World's largest community by number of packages on NPM
  3. Support for GPUs (GPU acceleration for optimal performance)

How can you utilize Liquid Carrot?

  1. Creating demos using artificial intelligence
  2. Organizing your data and implement machine learning to learn from a data set
  3. You can create environments with "fake" data (or very similar to the real thing) to train bots to be optimized
